模拟数据

前后端分离并行开发

当后端 API 没有编写完成时，前端无法进行调试，这就导致了前端会被后端阻塞的情况。FEZMOCK可以动态的生成文本、数字、布尔值、日期、邮箱、链接、图片、颜色等随机的模拟数据，让前端工程师不再依赖后端数据接口而独立进行开发。

高效可靠

模块化管理模拟数据

FEZMOCK支持不限于RESTFUL风格、$.ajax风格、及其它任何AJAX风格的API请求，配合数据模板及FEZ模块化管理模拟数据，通过配置可以真实的模拟任何业务场景研发、上线后的数据环境。
